Mobile Game Data Analysis
Overview

This project analyzes player behavior in a mobile game using SQL, Python, and Power BI.
The goal is to understand user activity, level difficulty, and gameplay performance.

Project Structure
dataset/   - Raw data (users, sessions, level_attempts)
sql/       - SQL queries for data analysis
python/    - Data analysis using Pandas
powerbi/   - Power BI dashboard

Key Analysis

New users by install date
Cumulate users by install date
Installed but never played users
Total sessions by country
Level fail rate
Daily active users
Average fail attempts by level
Top difficult level base on fail rate

Tools

SQL
Python (Pandas, NumPy)
Power BI
